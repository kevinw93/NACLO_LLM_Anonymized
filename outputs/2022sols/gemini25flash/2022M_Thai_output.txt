**(b) Arun's algorithm:**
Rules:
1.  Assign `1` to characters `c` that fit the `FC` slot in `V FC IC V` (preceded by V, followed by IC then V).
2.  Assign `1` if `c` is the last character.
3.  Assign `0` to all other characters.

This algorithm requires a look-ahead of 2 characters and a look-back of 1 character to determine the label for the current character (`FC`). A standard Finite State Transducer (FST) produces output for the current input. To implement look-ahead, the output for `char[i]` is often delayed until `char[i+k]` is processed. For `V FC IC V` where `FC` is `char[i]`, we need `char[i-1]`, `char[i+1]`, `char[i+2]`. This means the output for `char[i]` is determined when `char[i+2]` is read. This is a 2-delay FST.

*   The first two characters of the input are always labeled `0`.
*   For character `char_k` (starting from `k=2`):
    *   It determines the label for `char_{k-2}`.
    *   The label is `1` if `char_{k-2}=V`, `char_{k-1}=FC`, `char_k=IC`, `char_{k+1}=V` (implied by the next transition). Otherwise, it's `0`.
*   The "last character" rule overrides all other rules for the final character.

To draw this FST, we need states that remember the last two characters processed, and the current character decides the output for the character two steps ago.
Let `S(X, Y)` be a state where `X` was the character 2 steps ago, and `Y` was the character 1 step ago.
`S_empty`: Initial state.
`S_1st`: First character has been read.
`S(prev2, prev1)`: The state representing that `prev2` and `prev1` were the two most recent characters read (not including the current one).
`S_final`: Accepting state.

*   The `S_empty` and `S_1st` states handle the first two characters, always outputting `0` and buffering them. This means the actual pattern matching starts from the third character.
*   The `Delayed Output States` are a set of states `S_char(X)` where `X` represents the category of the character that was read two steps ago.
    *   `S_char(X)` is a conceptual state that splits into specific states based on the category of `X` (e.g., `S_V(X)`, `S_FC(X)`).
    *   Each transition `CurrentChar:label(X)` means that when `CurrentChar` is read, the label for the buffered `X` (2 characters ago) is output.
